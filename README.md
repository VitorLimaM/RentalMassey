# Property Rentals Platform

## Overview

This project is a front‑end web application that simulates a property rental platform where users can browse listings, view details, and interact with housing data in a realistic product environment.

The goal of the project was to replicate common SaaS patterns used in real estate platforms, focusing on usability, state handling, and scalable component structure rather than static pages.
