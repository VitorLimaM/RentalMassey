# 🏠 [RentalMassey] - Rental Platform Interface

> **A modern, high-performance rental platform interface built entirely with native web technologies.**


---

## 🎯 About the Project
This project is a **Vanilla Web Development** challenge focused on building a functional rental marketplace (for [houses/cars/equipment]). By avoiding heavy frameworks, this application demonstrates a deep understanding of core web fundamentals, resulting in a lightweight, fast, and dependency-free user experience.

### ✨ Key Features
* **Dynamic Listing:** Rental items rendered dynamically via JavaScript.
* **Smart Search & Filters:** Real-time filtering logic without page reloads.
* **Fully Responsive:** Optimized layout for Mobile, Tablet, and Desktop using CSS Grid and Flexbox.
* **Clean UI:** Modern aesthetic focused on usability and conversion.
* **No Dependencies:** 100% native code for maximum performance.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling, CSS variables, and smooth transitions.
* **JavaScript (ES6+):** DOM manipulation, event handling, and data filtering logic.
